# Style Constraints — Clean Professional Dashboard Guidelines

This document outlines the strict guidelines for maintaining a clean, professional, and standard dashboard experience for **Global News Pulse**. It forbids the introduction of mock-intelligence, security-monitoring, or mission-related roleplay terms and elements.

## 🚫 Forbidden Elements (Fluff & Roleplay)
Do NOT include any mock intelligence agency, surveillance, or mission-related text, placeholders, or styling decorations.
*   **No "Classification" meta**: Never add "CLASSIFICATION: UNCLASSIFIED", "EYES ONLY", "RESTRICTED AREA", or similar labels.
*   **No mock tracking numbers**: Never add "RECORD ID: VP-XXXXXX", "Unit 2024", or similar identifiers.
*   **No mock roles**: Never refer to users as "OPERATORS" or "ANALYSTS" (e.g., use "User Login", "Username", "Password" instead of "Operator ID", "Access Code", "Analyst Verification").
*   **No roleplay terms**: Avoid terms like "Geopolitical theater", "Surveillance nexus", "unlock the truth", "Dispatch", etc.

## 🟢 Allowed Elements (Professional Dashboard)
Use standard, plain, and professional business intelligence terms:
*   **Dossier/Briefing** -> Use "News Summary", "Digest", or "Report".
*   **Dispatch Index** -> Use "Topic Index" or "Sections".
*   **Key Actors** -> Use "Top Entities" or "Tracked Entities".
*   **Retrieving Dispatch** -> Use "Loading Report...".
*   **Security Clearance Required** -> Use "Sign In".
*   **Lockdown warnings** -> Do not include warning messages about system monitoring or lockdowns on the login page.
